# patient-health-analyzer
A Python tool that analyzes patient health indicators — BMI, blood pressure, 
and risk flagging — built as a first medical programming project.

## Description
This project is a command-line patient health data analyzer built in Python. 
It calculates and classifies BMI and blood pressure for multiple patients, 
counts abnormal indicators, and automatically flags patients as at risk when 
two or more indicators are abnormal. Built using core Python concepts — 
functions, loops, conditions, lists, and dictionaries.

## Features
- BMI calculation and WHO classification
- Blood pressure categorization based on ESC guidelines
- Automatic risk flagging for patients with 2 or more abnormal indicators
- Multi-patient analysis in a single run

## Technologies
- Python 3
- No external libraries required
